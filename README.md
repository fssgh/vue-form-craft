# vue-form-craft

基于 [vue](https://github.com/vuejs/vue) 和 [element-ui](https://github.com/ElemeFE/element) 实现的可视化表单设计器，使用了最新的前端技术栈，可以让表单开发简单而高效。


* [在线预览](https://hyl999.co:86/formDesign)
* [官网](https://hyl999.co:86/)

## 特性

* 可视化配置页面
* 简单高效的联动系统
* 一键预览配置的效果
* 一键生成配置json数据
* 提供自定义组件满足用户自定义需求
* 提供功能强大的高级组件
* 支持表单验证
* 快速获取表单数据
