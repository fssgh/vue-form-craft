# 简介

vue-form-craft是一款基于Vue 3.+的低代码表单工具，基于Element Plus UI库。

目的是为前端开发同学提供快速搭建表单、实现表单交互和数据收集的功能😄。

## 核心

vue-form-craft主要由 **表单设计器FormDesign** 、 **表单渲染器FormRender**两部分构成。

- FormDesign通过拖拽生成JsonSchema

- FormRender负责将JsonSchema渲染为表单
